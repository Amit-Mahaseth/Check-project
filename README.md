# CodeSherpa - Multi-Agent Developer Productivity Platform

A developer productivity tool powered by multi-agent AI, optimized for Indian developers.

## Features
- Multi-agent architecture (Review Monk, Sherpa, Doc Sync, etc.)
- GitHub Integration
- WhatsApp Integration
- Hindi/Multilingual Support
- AWS Bedrock (Claude Sonnet 3.5)

## Architecture
- Backend: Python 3.11 + FastAPI + AWS Bedrock
- Frontend: React 18 + Vite + TailwindCSS + shadcn/ui
- Agent Framework: LangChain + LangGraph
- Database: DynamoDB + Redis
